Testing PSIGNB (Packed SIGN Byte)

Test 1: Basic sign operations
a: [10, -20, 30, -40, 50, -60, 70, -80]
b: [1, -1, 0, 1, -1, 0, 1, -1]
result: [10, 20, 0, -40, -50, 0, 70, 80]
Expected: [10, 20, 0, -40, -50, 0, 70, 80]

Test 2: Boundary values
c: [127, -128, 0, 127, -128, 0, 127, -128]
d: [1, -1, 1, -1, 1, -1, 0, 0]

Expected: [127, -128, 0, -127, -128, 0, 0, 0]

Test 3: Memory operand
reg: [1, -1, 0, 1, -1, 0, 1, -1]
mem: [5, -5, 10, -10, 15, -15, 20, -20]
